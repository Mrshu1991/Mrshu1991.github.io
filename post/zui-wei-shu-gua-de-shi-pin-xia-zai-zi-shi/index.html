最为舒适的视频下载姿势

有一些做自媒体的朋友经常会需要用到很多视频素材，各大平台的视频必须要安装客户端才能缓存，缓存的文件格式还不通用。本文就介绍一个最为简单且下载速度极高的方式。

本方法基本上支持所有国内外主流的视频网站。

1. 安装Python

👉下载Python并安装。

2. 安装you-get

1️⃣ 同时按住win+R呼出运行窗口，输入cmd并回车；
2️⃣ 输入pip3 install you-get并回车；
3️⃣ 继续输入python -m pip install --upgrade pip并回车。

📢至此，所有准备工作都已完成。

3. 视频下载方法

打开命令行工具（win+r，输入CMD），输入you-get 视频链接并回车，即可；

🔔 如果你需要选择不同的清晰度视频进行下载，请参照以下方法👇

  • 打开命令行工具，输入you-get -i 视频链接
  • 根据需要的视频清晰度，选择对应的代码，如you-get --format=flv720 url，将该代码后的URL换成视频链接，输入，回车即可。
